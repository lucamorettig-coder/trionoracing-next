Note di design · scope MVP EVO-019

  1. Pagina dedicata (non modal): conferma scope EVO-019. Stesso form usato per /portale/admin/gare/nuova e /portale/admin/gare/[id]/modifica (cambia solo titolo + valori iniziali).
  2. Distinzione Descrizione vs Note: la Descrizione è visibile ai genitori (badge sun "Visibile ai genitori"), le Note interne sono riservate admin (badge neutral "Solo admin"). Campo Descrizione è nuovo nello schema Airtable.
  3. Multi-select Maestri: chip persone con foto/iniziali + cognome + qualifica FCI, popolato da getAllMaestriAttivi(). Pattern coerente con altre selection chip già in uso.
  4. Out of scope MVP: niente upload immagine copertina (R2), niente costo iscrizione, niente date apertura/chiusura iscrizioni. Tutti questi campi non esistono nello schema.
  5. Salvataggio: Server Action POST → redirect a /portale/admin/gare/[id]?success=created (o ?success=updated per modifica). Pattern coerente con EVO-017/018.
